# AI Resume Matcher

ابزار هوشمند تحلیل رزومه که با استفاده از هوش مصنوعی، رزومه شما را با شرح شغل مقایسه کرده و گزارش دقیقی از مهارت‌ها، نقاط ضعف و پیشنهادات بهبود ارائه می‌دهد.

## امکانات

- **آپلود رزومه PDF** - استخراج و تحلیل متن از فایل‌های PDF
- **تطبیق مهارت‌ها** - مقایسه مهارت‌های رزومه با نیازهای شغل
- **تحلیل شکاف‌ها** - شناسایی مهارت‌های مفقود و حوزه‌های ضعف
- **پیشنهادات بهبود** - راهکارهای عملی برای تقویت رزومه
- **سوالات مصاحبه** - تولید سوالات مرتبط بر اساس شرح شغل
- **امتیاز تطابق** - سنجش میزان هماهنگی رزومه با شغل (۰ تا ۱۰۰٪)
- **تاریخچه مبتنی بر نشست** - تاریخچه تحلیل هر کاربر فقط در مرورگر خودش ذخیره می‌شود
- **خروجی به زبان فارسی** - تمام نتایج تحلیل به زبان فارسی ارائه می‌شود

## فناوری‌های استفاده شده

- **بک‌اند**: Django 5.x
- **هوش مصنوعی**:
  - Hugging Face Inference API (Qwen/Qwen3-8B)
  - LangChain برای پردازش متن
  - Sentence Transformers (all-MiniLM-L6-v2) برای تبدیل متن به بردارها
- **پایگاه داده برداری**: ChromaDB
- **پردازش PDF**: PyMuPDF

## نصب و راه‌اندازی

1. کلون کردن مخزن:
```bash
git clone https://github.com/yourusername/ai-resume-matcher.git
cd ai-resume-matcher
```

2. ایجاد محیط مجازی:
```bash
python -m venv venv
source venv/bin/activate  # در ویندوز: venv\Scripts\activate
```

3. نصب وابستگی‌ها:
```bash
pip install -r requirements.txt
```

4. تنظیم متغیرهای محیطی:
```bash
cp .env.example .env
# فایل .env را ویرایش کنید
```

5. اجرای مایگریشن‌ها:
```bash
python manage.py migrate
```

6. راه‌اندازی سرور توسعه:
```bash
python manage.py runserver
```


## نحوه کار

1. **آپلود رزومه**: کاربر فایل PDF رزومه را آپلود می‌کند
2. **استخراج متن**: PyMuPDF متن فایل را استخراج می‌کند
3. **تقسیم متن**: LangChain متن را به بخش‌های کوچکتر تقسیم می‌کند
4. **ایجاد بردارها**: Sentence Transformers بخش‌ها را به بردار تبدیل می‌کند
5. **جستجوی برداری**: ChromaDB مرتبط‌ترین بخش‌های رزومه را پیدا می‌کند
6. **تحلیل هوش مصنوعی**: مدل زبانی بخش‌های مرتبط را با نیازهای شغل مقایسه می‌کند
7. **تولید گزارش**: نتایج شامل امتیاز، تحلیل مهارت‌ها و پیشنهادات است

## حریم خصوصی

- تاریخچه تحلیل هر کاربر فقط در نشست مرورگر خودش ذخیره می‌شود
- هیچ داده‌ای بین کاربران به اشتراک گذاشته نمی‌شود
- فایل‌های رزومه بلافاصله پس از پردازش حذف می‌شوند
- هیچ ذخیره‌سازی دائمی از رزومه‌های کاربران انجام نمی‌شود
